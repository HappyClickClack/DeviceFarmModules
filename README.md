## Коллекция идей и кода для работы с Android устройствами в тестовой ферме.

## scrcpy
Утилита, предназначенная для зеркалирования экрана Android-устройств на ПК (Windows, macOS, Linux) и управления ими с помощью мыши и клавиатуры.
Подробнее [тут](https://github.com/genymobile/scrcpy).

## Xpra
Программное обеспечение с открытым исходным кодом, предназначенное для удаленного запуска графических приложений (X11) и рабочих столов.
Подробнее [тут](https://github.com/Xpra-org/xpra/).

## Web доступ для управления Android устройствами
Комбинация scrcpy и Xpra или запуск scrcpy в Xpra.
Подробнее [тут](https://github.com/HappyClickClack/DeviceFarmModules/blob/main/xpra_scrcpy/README.md).

## Управление питанием устройств
Использование USB хабов с поддержкой PPPS для программного отключения портов с устройствами.
Подробнее [тут](https://github.com/HappyClickClack/DeviceFarmModules/blob/main/USB_power_control/README.md).

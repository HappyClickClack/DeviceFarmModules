## scrcpy
Утилита, предназначенная для зеркалирования экрана Android-устройств на ПК (Windows, macOS, Linux) и управления ими с помощью мыши и клавиатуры.
Подробнее [тут](https://github.com/genymobile/scrcpy).

## Xpra
Программное обеспечение с открытым исходным кодом, предназначенное для удаленного запуска графических приложений (X11) и рабочих столов.
Подробнее [тут](https://github.com/Xpra-org/xpra/).
